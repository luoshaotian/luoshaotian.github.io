将zotero变成科研的好工具

发表于2023-06-27|更新于2023-07-08
|字数总计:1.5k|阅读时长:4分钟|阅读量:

现在市面上有很多的文献管理工具，例如：zotero、endnote、Mendeley、Citavi（除了zotero笔者都没用过）。对于zotero这个软件，它最大的优势就是我喜欢用：），那么如何对他进行设置呢，这里本人认为难的地方会细讲，不难的地方一笔带过。

下载zotero 和 zotero connector

这一步去官网下载它会摆出两个选项供选择，将两个都选上。

注册zotero账号并登录

进入软件之后按照Edit-Preference-sync,里面有一个选项让你登录，没有就跳到官网注册一下

下载一些zotero辅助插件

登录上zotero的账户之后，分别在github下载一些插件，例如：zotfile，zotero-pdf-translate，zotero-scihub，zotero-doi-manager等等

  • zotfile:用来管理文件，可以对文件进行位置转移（方便找pdf文件），以及改名字等操作
  • zotero-pdf-translate:用来翻译pdf中的句子或者单词（使用效果不是很好），可以使用goldendict安装一些常用词典进行查找，体验感更好。
  • zotero-scihub：主要用于没有校园网的情况下，通过scihub下载文件，能解决90%的文献需求(首先需要找到文献的doi号)，处于校园网情况下，需要在学校官网查找相关“文献检索”软件的说明，里面可能会涉及到”resolver”的字眼，文章后面会详细解释
  • zotero-doi-manager:用于查找文件的doi号，一般使用long doi

文献检索

推荐使用google scholar来进行检索，步骤如下

  • 输入想要检索的内容，在搜索结果页面通过文献题目初筛，将需要的文献通过$\star$标志收藏（需要登录google账号）
  • 在将所有可能需要的文献收藏好后，在“我的图书馆”选项中选择全部导出为bibtex格式。
  • 在zotero的file选项中，选择import，选择生成的xxxx.bib文件，进行导入
  • 在安装好zotero-doi-manager的情况下，全选文献，右键，选择Manage-doi，获取long doi
  • 右键，find available pdf，下载所有选中条目的pdf
    • 情况一：下载了scihub插件，按照其官网的使用说明配置好后直接使用find available pdf 命令即可
    • 情况二：处于校园网环境下，通过搜寻学校官网关于文献检索信息的网页。一般会给出一个resolver，通过zotero的preference-advance，找到以下页面，将resolver填上即可使用find available pdf命令。对于一些找不到的文献可以考虑scihub插件下载，也可以双击条目跳转到相应网页下载（可能需要代理）

文献同步

文献同步的好处在于：当自己对文献做过笔记之后，通过文献同步的方法可以在不同的设备之间进行传递。能够保证自己的文件笔记不会消失。且不需要重复的去网上下载文献。对于没有这类需求的用户，完全可以放弃文献同步这一功能。

此处，给出三种文献同步的方法

通过zotero官方进行同步

zotero官方对每个用户分配了300MB的储存空间用于储存pdf文件，而对于用户的信息（除了pdf之外的所有信息）zotero是免费同步且没有限制的。因此，如果对pdf储存有需求，可能在官网花钱升级储存空间。只需要通过zotero官网的指引交钱即可不过真的非常的贵，不推荐，富哥随意。

通过坚果云+zotfile 进行同步

这个方法有很多的坑，需要十分注意。下面分两种情况讨论

  • 需要同步文献，但没有将文献统一管理需求的，你只需要注册坚果云账号，然后他会有专门的同步zotero文件夹的教程。按部就班操作即可
  • 需要同步文献，并且需要统一化管理文献（典型例子：在ipad下阅读文献），步骤如下
    • 第一步，Tools$\rightarrow$zotfile preference$,首先将“source Folder for Attaching New Files”的路径填成zotero的安装路径下的storage（例如：C:\User\username\Zotero\storage），再将下面的“Location of Files”选中你想要移动到的位置（例如：C:\User\username\Document\pdfs）.这一步的操作将zotero的pdf储存位置，从默认位置移动到自己想要的位置，方便后面查阅以及同步
    • 第二步，Prefernce $\rightarrow$ Advanced$\rightarrow$Files and Folders,将linked Attachment Base Directory 的路径改为上一步你自己想移动到的位置的路径，下面还有一个Data Directory Location，根据个人需求移动（这一步将zotfile移动pdf后的位置告诉zotero，让zotero能够找到pdf文件）
    • 第三步，同步坚果云，webdav。坚果云官网有教程

坚果云有一个缺陷就是有点贵，免费用户只有1GB的免费上传内存，这对于一个认真搞科研的人来说明显是不够的。购买它的内存，一年需要199元以上，为了节约开支，第三种方法诞生了！

通过服务器搭建owncloud进行同步

在阿里云腾讯云购买服务器，可以购买特价服务器，一年大概便宜50元左右。但是一年之后可能服务器会涨价，这个时候就需要重新购买新的服务器，重新搭建一次owncloud。推荐喜欢折腾的用户使用这种方法。

至于如何购买服务器，通过ssh链接服务器，搭建owncloud云盘的教程，网上都有。可能需要一点linux知识。linux的基础命令可以参考本人的这篇文章（只是转载了一些命令）。owncloud推荐使用docker进行搭建。详细搭建步骤可以参考owncloud的官方文档(P.S. 服务器可能会设置安全组，如果要搭建owncloud记得把TCP:8080端口打开，不然没有用)
